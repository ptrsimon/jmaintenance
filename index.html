jmaintenance

System recovery application for Debian GNU/Linux

Download this project as a .zip file Download this project as a tar.gz file

About

jmaintenance is a system recovery program that tries to fix common errors which might keep the system from booting. This program is for Debian GNU/Linux systems (tested on Wheezy 7.5, but other versions might work as well); do not use it on other distributions!

About project

This program is written for the JackOS Project, a Debian based, innovative GNU/Linux distribution.

Current status

jmaintenance is tested a lot, but it is still not ready for production systems. Currently it's in beta stage. You can help improve it by testing jmaintenance and reporting any bugs you find.

Documentation

The program is documented in its manual page, jmaintenance(8) (you can find it under doc/jmaintenance.8). More information is available on our wiki.
